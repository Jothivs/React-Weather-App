# React-Weather-App
         
                                                
This is a Weather App using React.

### Features

Shows the weather condition of any city including:
* type of weather ((e.g) cloudy, sunny, rainy,..etc.)
* weather value in Celsius
* latitudinal & longitudinal co-ordinates.
